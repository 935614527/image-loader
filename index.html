Image loader is a simple library that give you the ability to decouple the logic to fetch, load and cache images.

It does that away from the ui thread, and by caching images with two level in memory or in the sdcard if awailable.

Usage

The library is just a jar and can be easily included in every android project. You can even use maven.

Figure out more at the github page

Caching

The library implement 2 levels caching for you.

Concurrency

The library is fetching images with a thread running outside the ui thread

Proven

The library is tested as it is used on read application on the market. (For example imgsearch.)
